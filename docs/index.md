# Makerspace Wiki

Welcome. This is the reference guide for every tool and piece of equipment in the space — what it does, how to use it, what materials and settings it takes, and how to stay safe around it.

## How This Wiki Is Organized

Each category page below covers the same set of sections, in this order:

- **Machines** — one entry per physical machine
- **Software** — one entry per related software tool (where applicable)
- **Materials** — approved/prohibited materials (CNC & Laser Cutters only)
- **Tool Settings** — speeds, feeds, power/settings reference (CNC & Laser Cutters only)
- **Safety** — required training, PPE, and general safety rules for the category
- **FAQ** — common questions specific to that category

## Before You Use Anything

!!! danger "Read this first"
    If you have not been trained/certified on a specific machine, do not use it. Check the **Safety** section of the relevant category page, and ask a shop lead if you're unsure.
